## Deployment Preview

This project uses GitSlip for automatic preview deployments.

- Latest preview (any branch): https://toduu-ui-experiment.gitslip.com
- Production (main branch): https://main.toduu-ui-experiment.gitslip.com
- Branch-specific: https://{branch-name}.toduu-ui-experiment.gitslip.com

After creating a PR, tell the user their preview URL based on the branch name.
The preview may take 30-60 seconds to build. If they see a "building" page, it will auto-refresh when ready.
