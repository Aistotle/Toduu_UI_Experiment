# /ralph-loop — Autonomous Work Loop

> **PLUGIN WRAPPER** — This is documentation only. After reading, you MUST activate the plugin:
> → Run: `/ralph-wiggum:ralph-loop`

Start a Ralph Wiggum loop that runs autonomously in the current session, continuously working on tasks until stopped.

## Commands
- **Start loop**: `/ralph-wiggum:ralph-loop`
- **Cancel loop**: `/ralph-wiggum:cancel-ralph`
- **Get help**: `/ralph-wiggum:help`

## When to Use
- Long-running tasks that benefit from continuous autonomous work
- When you want Claude to keep iterating without manual prompting
- Batch processing or repetitive task sequences

## Usage
Start the loop:
```
/ralph-wiggum:ralph-loop
```

Cancel when done:
```
/ralph-wiggum:cancel-ralph
```
