# /analyze-codebase — Comprehensive Codebase Documentation

> **PLUGIN WRAPPER** — This is documentation only. After reading, you MUST activate the plugin:
> → Run: `/analyze-codebase:analyze-codebase`

Generate comprehensive analysis and documentation of the entire codebase.

## What It Does
- Analyzes project structure and architecture
- Documents key patterns and conventions
- Identifies dependencies and relationships
- Creates a high-level overview of the codebase

## When to Use
- Onboarding to a new codebase
- Creating or updating architecture documentation
- Understanding the full scope of a project
- Before major refactoring efforts

## Usage
Invoke the plugin:
```
/analyze-codebase:analyze-codebase
```
